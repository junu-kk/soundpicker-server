## soundpicker-server

전주 1초 듣고 음악 맞추는 게임 사운드피커의 서버입니다.

### Installation

```bash
$ pnpm install
```

### Running the app

```bash
# development
$ pnpm start

# watch mode
$ pnpm start:dev

# production mode
$ pnpm start:prod
```

### Test

```bash
# unit tests
$ pnpm test

# e2e tests
$ pnpm test:e2e

# test coverage
$ pnpm test:cov
```
